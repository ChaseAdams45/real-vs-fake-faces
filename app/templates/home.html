AI Fake vs. Real Face Detector

About the Project

For this project, I decided to try to create a model that can detect whether a face is real or fake. I thought this would be interesting to try and tackle because in one of my recent ethics courses, we discussed deepfakes and how they are harmful in spreading misinformation. I did not quite attain the results I wanted, given limited time to train and iterate to try to improve the model, but the final model I created was still able to do some learning shown by the confusion matrix. I had originally tried training on a smaller set size with lower quality pictures, but my final model was trained on 223 416x416 images.
